# 📊 Análise de Dados do ENEM 2019

Projeto desenvolvido para a disciplina de Ciência de Dados da Universidade Federal do Amazonas (UFAM), com foco em análise exploratória de dados do ENEM 2019 utilizando Python e bibliotecas de Data Science.

## 📌 Objetivo

O objetivo deste projeto é explorar e analisar os microdados do ENEM 2019, buscando identificar padrões, distribuições e relações entre variáveis socioeconômicas e o desempenho dos candidatos em diferentes regiões do Brasil.

## 🛠️ Tecnologias Utilizadas

* Python 3.11.7
* Pandas
* NumPy
* Seaborn
* Matplotlib
* SciPy
* Scikit-learn
* Jupyter Notebook

## 📊 Análises Realizadas

* Exploração inicial dos dados
* Limpeza e tratamento de dados
* Estatísticas descritivas
* Distribuição de notas do ENEM
* Comparação entre cidades e estados
* Visualização gráfica de variáveis
* Correlação entre fatores socioeconômicos e desempenho

## ▶️ Como Executar o Projeto

Clone o repositório:

```bash
git clone git@github.com:matheusluix/analise-de-dados-Enem-2019.git
```

Entre na pasta do projeto:

```bash
cd analise-de-dados-Enem-2019
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o Jupyter Notebook:

```bash
jupyter notebook
```

## 📁 Dataset

Os dados utilizados neste projeto estão no formato `.parquet` e foram obtidos a partir dos microdados disponibilizados pelo INEP.

* Microdados ENEM: https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados
